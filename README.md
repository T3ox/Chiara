# FolderOrganizer

Repository organizzato in due aree:

- `site/`: frontend landing in React + Vite
- `programma/`: script FolderOrganizer + prompt JSON + `.env`

## Avvio sito (React + Vite)

```powershell
cd site
npm install
npm run dev
```

Apri `http://127.0.0.1:5173`.

## Avvio programma

```powershell
cd programma
python FolderOrganizer.py
```
