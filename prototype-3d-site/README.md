# FolderOrganizer 3D Prototype

Prototipo isolato per sperimentare una nuova esperienza 3D senza toccare il sito principale.

## Avvio locale

```powershell
cd prototype-3d-site
npm install
npm run dev
```

Server locale previsto: `http://localhost:5174`

## Note

- Tutto il codice del prototipo e confinato in `prototype-3d-site`.
- Il sito principale in `site/` e la landing statica in `site_html/` non vengono modificati.
