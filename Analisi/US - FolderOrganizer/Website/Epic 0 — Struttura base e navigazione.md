# Epic 0 — Struttura base e navigazione

### US-0.1 — Layout e navigazione principale

**Come** visitatore

**Voglio** vedere header/footer coerenti e un menu chiaro

**Così** posso muovermi tra Home, Prezzi, Account, Privacy

**Acceptance criteria**

- Header con link: Home, Prezzi, Privacy, Account (o “Accedi” se non loggato)
- Footer con: contatti, privacy, termini, link social (se presenti)
- Evidenza pagina attiva nel menu
- Mobile: menu hamburger
